# 🏦 ATM Simulator (Java)

A console-based ATM (Automated Teller Machine) system built with Java. This project demonstrates the core principles of **Object-Oriented Programming (OOP)**, including encapsulation and conditional logic, to manage banking transactions.

---

### ✨ Features
* 💰 **Check Balance:** View the current account balance instantly.
* 💵 **Deposit Funds:** Add money to the account with real-time balance updates.
* 💸 **Withdraw Funds:** Securely withdraw money (includes insufficient funds check).
* 🚪 **Secure Exit:** Safely terminate the session.

### 🚀 How It Works
The system follows a simple transaction loop:
1. **User Input:** The user chooses an option from the menu using the `Scanner` class.
2. **Validation:** For withdrawals, the system checks if the requested amount is available in the balance.
3. **Update:** The balance is updated dynamically after every successful deposit or withdrawal.
